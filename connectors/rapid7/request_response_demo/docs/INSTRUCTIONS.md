# __Description__

  Connector for Surface Command that imports Demo Device and User data

# __Overview__

  This is a demo Connector that uses https://reqres.in to get example User and Device data

  It is to be used as a reference when developing Connectors with Rapid7 Surface Command

# __Documentation__

  ## __Setup__

  To configure this Connector you will need to request a FREE API KEY at [https://reqres.in/](https://reqres.in/),
  add a Connector Profile and update the Settings
