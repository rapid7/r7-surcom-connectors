# __Description__

  Connector for Paessler PRTG Network Monitor

# __Overview__

  PRTG is a unified monitoring tool that can monitor almost any object that has an IP address. It consists of the PRTG core server and one or more probes, which collect data and monitor processes on devices via sensors.

  This connector synchronizes the device details from PRTG into the Rapid7 Platform.

# __Documentation__

  The connector requires `Base URL`, `Username` and `Passhash` for fetching the API.

  To generate the Passhash, refer to the documentation [here](https://www.paessler.com/manuals/prtg/my_account#apiaccess)
