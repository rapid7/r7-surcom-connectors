# __Description__
  Connector for EZO AssetSonar ITAM

# __Overview__
  EZO AssetSonar is a cloud-based IT Asset Management (ITAM) and Asset Tracking platform that helps organizations monitor and manage their IT and physical assets throughout their entire lifecycle from procurement to retirement.

  The AssetSonar connector imports detailed asset information into Surface Command, including Fixed Assets, Locations, Members, Groups, and Subgroups. This enables centralized visibility and continuous monitoring of your organization's asset intelligence within Rapid7.

# __Documentation__
  To configure the EZO AssetSonar connector, you will need your organization’s `API Base URL` and `API Access Token`.

  ### To generate an API Access Token

  1. Log in as the Account Owner to access API configuration settings.
  2. Navigate to the sign-in page in your AssetSonar account.
  3. Click the **Settings** icon, then select the **Integrations** tab. At the top of this page, locate the **API Integration** section.
  4. Ensure that the status is `Enabled`.
  5. Once enabled, **generate the Access Token**. This token will be used to authenticate all API requests.
