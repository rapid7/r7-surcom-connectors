# __Description__
  Connector for SecurityScorecard

# __Overview__

  SecurityScorecard is a cybersecurity risk rating platform that provides continuous, external assessments of an organization's security posture across key risk factors.

  This connector synchronizes security ratings, issue findings, and attack surface intelligence from SecurityScorecard into Surface Command.

# __Documentation__
  This connector requires the Base URL to the SecurityScorecard API which is normally `https://api.securityscorecard.io/`, the Company Domain Name and an API Key

  ### To Get the Company Domain Name
  1. Login to the SecurityScorecard UI
  2. Expand your profile by clicking the user icon in the top right
  3. Copy the domain name, for example `rapid7.com`

  ### To Generate an API Key
  * Refer to the SecurityScorecard [documentation](https://securityscorecard.readme.io/reference/quickstart)
