# __Description__

  Connector for Datto RMM

# __Overview__

  Datto RMM is a cloud-based Remote Monitoring and Management (RMM) platform that enables service providers and IT teams to securely monitor, manage, and support endpoints.

  This connector synchronizes endpoint details from Datto RMM into the Rapid7 Platform.

# __Documentation__

  This connector requires an API URL, Key and Secret for authentication.

  To active the API and generate a Key and a Secret in Datto RMM, follow the steps in their [documentation](https://rmm.datto.com/help/en/Content/2SETUP/APIv2.htm) in the **How to** section
