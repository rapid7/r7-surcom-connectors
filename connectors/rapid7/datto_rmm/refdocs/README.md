### To Copy the Latest Swagger JSON

- Go to the connector refdocs
- Click on the below link to download the latest Swagger JSON. Please always ensure you have the latest version.
  "https://syrah-api.centrastage.net/api/v3/api-docs/Datto-RMM"
- Note: `refdocs/swagger.json` file `bios.releaseDate` `AttachedDevice.driverFileLastModified` property type is
  overwritten to `integer`, because to handle the response `bios.releaseDate` `AttachedDevice.driverFileLastModified`
  property value can be integer.
