# __Description__
  Connector for Nautobot network automation and source-of-truth platform.

# __Overview__

  Nautobot is an open source Network Source of Truth and Network Automation Platform built as a web application atop the Django Python framework.

  This connector integrates device inventory, network topology, IP address management, and virtualization data with the Rapid7 Platform.

# __Documentation__

  ## Configuration

  To configure the Nautobot Connector, you must provide the following settings:

  | Setting | Description |
  |---------|-------------|
  | **Base URL** | The base URL for your Nautobot instance (e.g., `https://nautobot.example.com`) |
  | **API Token** | API Token for authentication |

  To generate an API key (API token), see the [Nautobot API Token documentation](https://docs.nautobot.com/projects/core/en/stable/user-guide/platform-functionality/users/token/).
