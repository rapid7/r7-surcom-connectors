# __Description__

  Connector for Veracode

# __Overview__
  Veracode is a comprehensive software security platform that automates application security testing through static, dynamic, and software composition analysis to detect and remediate vulnerabilities.
  This connector ingests Applications and static and dynamic analysis findings into the Rapid7 Platform.
  
# __Documentation__

  Configuring the Veracode Connector involves identifying the appropriate `API URL`, and generating the necessary `API ID` and `API KEY`.

  To identify the correct Veracode API URL for your region and environment, refer to the following documentation:
  [API URL Documentation](https://docs.veracode.com/r/IP_addresses#all-veracode-products---apis-and-integrations)

  To generate the **API ID** and **API Key** refer to the following documentation:
  [API Credentials Generation Documentation](https://docs.veracode.com/r/c_api_credentials3#generate-api-credentials)
