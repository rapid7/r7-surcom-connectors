# __Description__
  Connector for Cisco Intersight

# __Overview__

  Cisco Intersight is a cloud operations platform that consists of optional, modular
  capabilities of advanced infrastructure, and workload optimization. Cisco Intersight
  infrastructure services include the deployment, monitoring, management, and support
  of your physical and virtual infrastructure. It supports Cisco Unified Computing
  System (Cisco UCS) and Cisco HyperFlex hyperconverged infrastructure (HCI), and other
  third-party Intersight-connected targets

  This connector synchronizes resource details from Intersight into Surface Command.

# __Documentation__

  To configure the Cisco Intersight connector, you need to select the appropriate
  `Region` based on your Cisco Intersight account and provide your `Key ID` and
  `Private Key`.

  For instructions to generate an API key, follow the steps described [here].(https://developer.cisco.com/docs/intersight/authentication/#generating-api-keys)
