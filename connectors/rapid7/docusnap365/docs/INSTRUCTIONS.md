# __Description__

  Connector for Docusnap365

# __Overview__

  Docusnap365 helps with IT inventory, analysis and audits. This automatically creates network maps, reports, operational and disaster recovery manuals.

  This connector synchronizes your Docusnap365 inventory with the Rapid7 Platform.

# __Documentation__

  This connector requires an API key from the Docusnap365 Platform, with at least Read permissions.

  For guidance on creating an API key, please refer to the documentation [here](https://api-doc.docusnap.com/reference/first-steps).
  If you set an expiration date on the API key, be sure to set a reminder in your calendar to avoid failure later.

  In case of any difficulties with the API key permissions, you can test the key in
  Docusnap's [online API explorer](https://api-doc.docusnap.com/reference/systemslist),
  by pasting the API key into the "Header" field and using the "Try It!" button.
