# __Description__

  Connector for Revivn

# __Overview__

  Revivn is a hardware lifecycle platform that helps companies manage their IT assets.
  
  This connector integrates asset inventory and end-of-life disposition status from Revivn with the Rapid7 Platform.

# __Documentation__

  This connector requires a Client ID and Secret to be configured.

  To get these settings, follow the below steps:

  ### Client ID and Secret
  1. Login to the Revivn App
  2. Generate Oauth2 Credentials from the API Section
  3. Copy and save the generated **Client ID** and **Secret**
