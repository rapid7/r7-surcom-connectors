# __Description__

  Connector for OneLogin

# __Overview__

  OneLogin is a cloud-based Identity and Access Management (IAM) platform that helps organizations securely manage user access with Single Sign-On (SSO), Multi-Factor Authentication (MFA), and easy user lifecycle management.

  This connector synchronizes the list of OneLogin users, groups, and applications into the Rapid7 Platform.

# __Documentation__

  To configure the OneLogin Connector, you must provide a URL, Client ID, and Client Secret.

  ### To generate the Client ID and Client Secret
  Refer to the OneLogin [documentation](https://developers.onelogin.com/api-docs/2/getting-started/dev-overview)
